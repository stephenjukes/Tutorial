CSS Selectors

CSS Selectors are ways that you can use CSS to identify html elements for styling. We will have used 'tag' selectors in previous lessons, eg: h1 { color: blue; } where h1 is the tag, and possibly also 'id' and 'class' selectors, (these are repeated here if this is the case). These do not represent all selectors, but a sample of the main types that are used.
