HTML

Introduction

  • HTML stands for Hyper Text Markup Language.
  • It is responsible for providing the content and structure for your web page
  • It consists of elements which implies what its content represents
  • Different elements are given their own default styling to help fulfil their intended role, eg:
    • strong elements are bold and naturally appears as part of a sentence
    • list items (like this one) are indented, preceded by a bullet point and given its own line
  • Default styling can be overridden, but the correct element should still be used to provide meaning to the content it represents

Basic elements

Elements generally consist of an opening and closing tag, with the displayed content in between. The opening and closing tag share the same name, but the closing tag is preceded by a /. Tags are themselves enclosed between angle brackets.

Attributes

Elements can also have attributes, which provide further information about the element. Attributes are found within the opening tag, after the tag name, and consist of:

  • the attribute name
  • an = symbol
  • a value enclosed by speech marks

Self-closing tags

Some elments are self-closing, which means that they do not require a closing tag. These elments do not have 'content' in the same way as elements that require a closing tag, but content and other information will generally be provided via one or more attributes.

Nested elements

Elements can be formatted inside other elements by nesting. This means that both opening and closing tags of a "child" element will appear between its "parent's" opening and closing tags
