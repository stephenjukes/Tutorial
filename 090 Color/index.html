Color

Introduction

  • Hex colors are created using a # symbol, followed by 6 digits
  • The digits represent 3 pairs of digits, with each pair representing the colors: red, green and blue, in that order
  • Each pair represents how much of each primary color should be included as part of the composite color, and should be thought of as as mixture of light, as opposed to of pigments such as paint. Hence:
    • red and green produces yellow, not brown
    • no contribution from each color produces black (absence of light)
    • full contribution from each color produces white (natural light)
  • Each pair is written in hexidecimal, (base 16)
    • Being larger than the decimal system we are used to working with, the hexidecimal system requires further digits, and the letters 'a' to 'f' are used for this purpose
